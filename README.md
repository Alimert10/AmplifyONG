# AmplifyONG - Online Resource Library

## Despre Proiect
Acest proiect a fost dezvoltat in cadrul unui hackathon organizat de Facultatea de Automatica si Calculatoare. Echipa noastra, formata din patru membri, a creat o platforma digitala menita sa sprijine ONG-urile prin oferirea accesului la o biblioteca online de resurse.

## Rolul meu in echipa
M-am ocupat de dezvoltarea partii de **front-end**, asigurandu-ma ca interfata este intuitiva, moderna si usor de navigat pentru utilizatorii finali.

## Functionalitati
* **Interfata Responsiva**: Design adaptabil pentru diferite dispozitive.
* **Biblioteca de Resurse**: Sectiuni organizate pentru documente, ghiduri si materiale utile.
* **Navigare Facila**: Meniu structurat si pagini optimizate pentru o experienta de utilizare placuta.

## Tehnologii Utilizate
* **HTML5**: Structura semantica a paginilor web.
* **CSS3**: Stiluri personalizate si layout-uri moderne.
* **JavaScript**: Elemente interactive pentru imbunatatirea experientei utilizatorului.

## Context
Proiectul a fost realizat intr-un interval de timp limitat, punand accent pe colaborarea eficienta in echipa si livrarea unui prototip functional pentru sectorul non-guvernamental.
